What it does: Controls how much the video must change before creating a new slide. Think of it like adjusting how "different" two scenes need to be.

  • Lower values (0.1-0.3): More sensitive - captures subtle changes like camera movements, zooming, or small scene shifts. Great for detailed presentations or when you want many slides.
  • Medium values (0.3-0.5): Balanced - detects clear scene changes like new locations, major camera angle changes, or significant content shifts. Good for most videos.
  • Higher values (0.5-0.8): Less sensitive - only captures major changes like completely different scenes or locations. Use when you want fewer slides with only the most important moments.

What it does: Sets the minimum time (in frames) that must pass between detected scene changes. This prevents creating too many slides from rapid changes or camera shakes.

  • Lower values (10-20): Allows slides to be created closer together. Use for videos with quick cuts, fast-paced content, or when you want to capture rapid changes.
  • Medium values (30-50): Balanced spacing - prevents duplicate slides while still capturing important moments. Good for most videos (default: 30).
  • Higher values (60-120): Creates more space between slides. Use for longer presentations, slower-paced videos, or when you want to avoid too many similar slides.
